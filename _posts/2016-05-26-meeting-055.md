### A. Infinite Sequence

[문제링크](http://codeforces.com/contest/675/problem/A)

등차수열의 첫항, 목표값 그리고 공차가 주어질 때, 목표값이 수열에 포함되는 지 여부

간단히 이렇게 하면 될 것 같지만,
```
    while (true) {
        if (begin == target) {
            return true;
        }
        begin += jump;

        if (0 < jump && target < begin) break;
        if (jump < 0 && begin < target) break;
        if (jump == 0) break;
    }
    return false;
```
이 방법은 꽤 오래 걸린다.<br>
worst case 는 시작이 -10^9, 공차가 1, 목표값이 10^9 인 경우이다.
