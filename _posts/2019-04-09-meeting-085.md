## 1

문제 요약: 4를 반드시 포함하는 입력 K가 주어질 때 다음 조건을 만족하는 두 양수 A, B 를 출력하라 

- A, B 는 4를 포함하지 않음
- A + B = K

입력 예시
```
3
4
940
4444
```
출력 예시
```
Case #1: 2 2
Case #2: 852 88
Case #3: 667 3777
```

## 2

문제 요약: N * N 크기의 미로에서 최단 경로로 이동한 경로가 입력으로 주어질 때,
동일한 경로를 포함하지 않는 다른 최단 경로를 출력하라.

동일한 경로란: 임의의 cell A -> B 의 이동 쌍이 겹치지 않음을 말함

입력 예시
```
2
2
SE
5
EESSSESE
```
출력 예시
```
Case #1: ES
Case #2: SEEESSES
```
